\subsection{Detailed Description}
Support and F\-A\-Q\-: visit \href{http://www.atmel.com/design-support/}{\tt Atmel Support}

See \hyperlink{ioport_quickstart}{Quick start guide for the common I\-O\-P\-O\-R\-T service}.

This is common I\-O\-P\-O\-R\-T service for G\-P\-I\-O pin configuration and control in a standardized manner across the M\-E\-G\-A, M\-E\-G\-A\-\_\-\-R\-F, X\-M\-E\-G\-A, U\-C3 and A\-R\-M devices.

Port pin control code is optimized for each platform, and should produce both compact and fast execution times when used with constant values.\hypertarget{group__usart__group_dependencies}{}\subsection{Dependencies}\label{group__usart__group_dependencies}
This driver depends on the following modules\-:
\begin{DoxyItemize}
\item \hyperlink{group__sysclk__group}{System Clock Management} for clock speed and functions.
\end{DoxyItemize}\hypertarget{group__ioport__group_ioport_modes}{}\subsection{I\-O\-P\-O\-R\-T Modes}\label{group__ioport__group_ioport_modes}
For details on these please see the X\-M\-E\-G\-A Manual.

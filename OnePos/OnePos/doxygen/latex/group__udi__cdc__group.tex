\subsection{Detailed Description}
Common A\-P\-Is used by high level application to use this U\-S\-B class.

These routines are used to transfer and control data to/from U\-S\-B C\-D\-C endpoint.

See \hyperlink{udi_cdc_quickstart}{Quick start guide for U\-S\-B device Communication Class Device module (U\-D\-I C\-D\-C)}.
